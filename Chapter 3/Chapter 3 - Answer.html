Detailed Explanations

Question 1

Correct Answer: B (Using a code review process)

Explanation: Logic bombs are code inserted into applications that execute under specific conditions. Code review processes can identify suspicious code, unauthorized functionality, or time-based triggers that characterize logic bombs.

Question 2

Correct Answer: C (Pop-ups demanding a ransom)

Explanation: Pop-ups demanding ransom payments are characteristic of ransomware, not rootkits. Rootkits are designed to remain hidden and provide backdoor access, not to extort money from users through visible ransom demands.

Question 3

Correct Answer: A (A keylogger)

Explanation: A keylogger captures all keyboard input, including passwords when they are typed. This would allow a student to obtain the staff member's credentials without their knowledge.

Question 4

Correct Answer: A (Command and control)

Explanation: TCP port 6667 is the standard port for Internet Relay Chat (IRC), which is commonly used for command and control communications by malware.

Question 5

Correct Answer: D (A backdoor)

Explanation: Software that allows remote access to systems is classified as a backdoor. The most precise classification for software providing unauthorized remote access is a backdoor.

Question 6

Correct Answer: A (Consuming resources)

Explanation: Bloatware primarily impacts systems by consuming system resources (disk space, memory, CPU cycles) without providing benefit to users.

Question 7

Correct Answer: C (Spyware)

Explanation: Spyware is specifically designed to gather information about users, including browsing habits and system information.

Question 8

Correct Answer: D (Different vendors use different names for malware packages.)

Explanation: Different antivirus vendors often use different naming conventions for the same malware. There is no standardized naming system.

Question 9

Correct Answer: D (Keyboard and other input from the user)

Explanation: Software keyloggers capture keyboard input and may also capture other input such as mouse movements, touchscreen inputs, and credit card swipes.

Question 10

Correct Answer: C (Ransomware)

Explanation: A message demanding payment in Bitcoin for encrypted files is the classic signature of ransomware.

Question 11

Correct Answer: B (Mount the drive on another system and scan it that way.)

Explanation: Rootkits are designed to hide from detection by hooking into the operating system. Mounting the drive on a trusted, clean system prevents the rootkit from hiding its presence.

Question 12

Correct Answer: C (A logic bomb)

Explanation: A logic bomb is code that executes when specific conditions are met (in this case, the developer not being employed).

Question 13

Correct Answer: B (Wipe the drive and reinstall from known good media.)

Explanation: The most reliable method is to wipe the drive completely and reinstall from known good media or restore from a clean backup.

Question 14

Correct Answer: B (How they spread)

Explanation: The key difference between worms and viruses is their propagation method. Worms spread themselves automatically without user interaction, while viruses require user action to spread.

Question 15

Correct Answer: B (Open the file using a text editor to review the code)

Explanation: Python is an interpreted language with source code that can be read directly. Opening the Python file in a text editor allows direct examination of the code logic.

Question 16

Correct Answer: B (Preventing downloads from application stores)

Explanation: Trojans are malware disguised as legitimate software that users voluntarily download and run. Preventing downloads from application stores directly addresses the primary infection vector for Trojans.

Question 17

Correct Answer: C (A worm)

Explanation: When malware exploits a network vulnerability to spread across multiple systems, this indicates worm behavior.

Question 18

Correct Answer: D (Bloatware)

Explanation: Unwanted programs pre-installed by manufacturers that load at startup are classic examples of bloatware.

Question 19

Correct Answer: A (A bot)

Explanation: Malware that connects to command and control systems for remote management are called bots. Multiple bots under centralized control form a botnet.

Question 20

Correct Answer: C (A trojan)

Explanation: When a user manually opens a file that appears innocent but contains malicious content, this is characteristic of a Trojan.

üîí Ransomware

Definition: Malware that encrypts files and demands payment
IoCs: C&C traffic, file encryption, ransom notices, data exfiltration, lateral movement
Mitigation: Offline backups, email security, user training, endpoint protection

üêé Trojans

Definition: Malware disguised as legitimate software
IoCs: Malware signatures, C&C communications, suspicious file creation
Mitigation: User awareness, application control, EDR tools

üêõ Worms

Definition: Self-propagating malware, spreads automatically
IoCs: Network scanning, vulnerability exploitation, automated spreading
Mitigation: Network segmentation, patching, firewalls, IPS

üïµÔ∏è Spyware

Definition: Gathers information about users and systems
IoCs: Data exfiltration, browser injection, disguised processes
Mitigation: Anti-spyware tools, application control, user awareness

üì¶ Bloatware

Definition: Unwanted pre-installed software
IoCs: Resource consumption, startup programs, vendor software
Mitigation: Uninstallation, clean OS images, deployment standards

ü¶† Viruses

Definition: Self-replicating malware requiring user action
IoCs: File modifications, replication behavior, payload activation
Mitigation: Antivirus, user awareness, system updates, behavioral analysis

‚å®Ô∏è Keyloggers

Definition: Capture keyboard and input device data
IoCs: Input capture, credential theft, data exfiltration

üí£ Logic Bombs

Definition: Code that executes under specific conditions
IoCs: Conditional code, time-based triggers, insider threats
Mitigation: Code review, secure development, change management

üîê Rootkits

Definition: Provide backdoor access while hiding presence
IoCs: System hooks, hidden processes, persistence mechanisms
Mitigation: External scanning, integrity checking, system rebuild

ü§ñ Bots & C&C

üîë Key Malware Characteristics

  • Ransomware: Encrypts files, demands payment
  • Trojan: Disguised as legitimate software
  • Worm: Spreads automatically via network
  • Virus: Needs user action to spread
  • Spyware: Gathers user/system info
  • Keylogger: Captures input data
  • Rootkit: Hides presence, provides backdoor
  • Logic Bomb: Executes on specific conditions
  • Bloatware: Consumes resources, unwanted

üí° Exam Success Tips

Scenario Analysis:
  • Look for payment demands ‚Üí Ransomware
  • Disguised software ‚Üí Trojan
  • Network spreading ‚Üí Worm
  • Captured keystrokes ‚Üí Keylogger

Common Traps:
  • NOT/EXCEPT questions - choose wrong answer
  • RATs vs. legitimate remote tools
  • Spyware vs. Bloatware distinction
  • Virus vs. Worm propagation methods
  • False positives in detection tools
